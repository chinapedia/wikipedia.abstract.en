COLCHESTER CARRIAGE SERVICING DEPOT is a stabling point located in Colchester, Essex, England. The depot is situated on the Great Eastern Main Line and is near Colchester station.

The depot code is CR.


Present

As of 2016, the depot has no allocation. It is, instead, a stabling point for Greater Anglia Class 153/156 Sprinters,[1] Class 170 Turbostars, Class 321[2] and Class 360 EMUs.[3]
