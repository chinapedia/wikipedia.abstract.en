CORKERHILL CARRIAGE SERVICING MAINTENANCE DEPOT is located in Corkerhill, Glasgow, Scotland. The depot is situated on the Paisley Canal Line and is to the east of Corkerhill station.

The depot code is CK.


Allocation

As of 2019, Corkerhill is responsible for the maintenance and cleaning of all 43 ScotRail Class 156 Super Sprinters. It is also a stabling and cleaning point for 18 ScotRail Class 158 Express Sprinters, Class 314 EMUs, Class 380 Desiros, First TransPennine Express Class 185 Desiros & First TransPennine Express Class 350 Desiros.
