COWLAIRS CARRIAGE MAINTENANCE DEPOT is a stabling point located in Cowlairs, Glasgow, Scotland. The depot is situated on the Edinburgh and Glasgow Railway.

The depot code is GC.


Allocation

As of 2016, there is no allocation. It is, instead, a stabling point for ScotRail Class 156 Sprinters.[1]
