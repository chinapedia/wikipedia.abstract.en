WORCESTER LIGHT MAINTENANCE DEPOT is a traction maintenance depot located in Worcester, Worcestershire, England. The depot is situated near Worcester Shrub Hill station.

The depot code is WS.


Present

As of 2016, the depot has no allocation. It is, instead, a stabling point for London Midland Class 150/153 Sprinters,[1] Class 170 Turbostars and Class 172 DMUs.
