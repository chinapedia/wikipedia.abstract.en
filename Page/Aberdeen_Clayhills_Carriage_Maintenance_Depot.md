ABERDEEN CLAYHILLS CARRIAGE MAINTENANCE DEPOT is a stabling point located in Aberdeen, Scotland. The depot is situated on the Denburn Valley Line and is located near Aberdeen station.

The depot code is AC.


Present

As of 2017, Clayhills depot is used for maintaining and stabling long distance locomotive hauled services, more specifically, London North Eastern Railway Class 43 HSTs[1]sets, Mark 3 carriages and Caledonian Sleeper Mk3 coaches hauled by Class 73 and 66 or Class 67 locomotives, ScotRail Class 158 Express Sprinters and Class 170 Turbostars.[2]
