NORWICH STATION CARRIAGE SERVICING DEPOT is a stabling point located in Norwich, Norfolk, England. The depot is situated on the Great Eastern Main Line and is near Norwich Station.

The depot code is NR.


Present

As of 2016, the depot has no allocation. It is, instead, a stabling point for East Midlands Trains Class 158 Express Sprinters[1] and Greater Anglia Class 153/156 Sprinters[2] and Class 170 Turbostars.
