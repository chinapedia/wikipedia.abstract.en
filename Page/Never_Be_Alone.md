_NEVER BE ALONE_ is an upcoming Philippine television drama series to be broadcast by GMA Network starring Carla Abellana, Sanya Lopez, and Sunshine Dizon. It is set to premiere on 2018.

Main Cast

-   Carla Abellana as Carmela
-   Sanya Lopez as Dorothy
-   Sunshine Dizon as Miranda
