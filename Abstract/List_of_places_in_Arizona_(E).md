This list of current cities, towns, unincorporated communities, counties, and other recognized places in the U.S. state of Arizona, which start with the letter E. This list only includes populated areas; it excludes physical features, ghost towns, and historical places. This list also includes information on the number and names of counties in which the place lies, its lower and upper ZIP code bounds, if applicable, its U.S. Geological Survey (USGS) reference number(s) (called the GNIS), class as designated by the USGS, and incorporated community located in (if applicable).
