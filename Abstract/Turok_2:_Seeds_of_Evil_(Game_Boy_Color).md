_TUROK 2: SEEDS OF EVIL_ is an action-platform video game developed by Bit Managers and published by Acclaim Entertainment. It was released for the Game Boy Color handheld game console in 1998 alongside its Nintendo 64 counterpart. The game is backward compatible with the original Game Boy in monochrome mode. Although the game is set in the same fictional universe as its counterpart, it features a different storyline.
