_MONOPOLY HERE AND NOW_ is an iOS video game adaption of the board game of the same name, which itself is a version of the classic board game Monopoly. It was developed by EA Mobile and Hasbro. The game marked the franchise's debut into the iOS market,[1] and was launched on May 21, 2009.[2] The game was also scheduled for release on Nokia N-Gage.[3] The game was added to the Pogo.com platform in 2009.[4]
