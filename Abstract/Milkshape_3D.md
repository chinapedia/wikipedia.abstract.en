MILKSHAPE 3D (MS3D) is a shareware low-polygon 3D modeling program created by Mete Ciragan. It is used mainly for compiling models for _Half-Life_, _Blockland_, _The Sims 2_, _The Sims 3_, _Rock Raiders_, and other sandbox video games.^([_citation\ needed_]) It is also used to create models for a large number of indie games, Milkshape 3D's repertoire of export capabilities has been extended considerably, due to the efforts of both its creator and the community around it, and it is now able to be used for most games today, so long as an exporter for the required format is available.
