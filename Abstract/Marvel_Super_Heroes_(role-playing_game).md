_MARVEL SUPER HEROES_ (MSHRPG) is a role playing game set in the Marvel Universe, first published by TSR as _Marvel Super Heroes: The Heroic Role-Playing Game_ under license from Marvel Comics in 1984.[1] In 1986, TSR published an expanded edition, entitled the _Marvel Superheroes Advanced Game_. Jeff Grubb designed both editions, and Steve Winter wrote both editions. Both use the same game system.
